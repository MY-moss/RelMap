# AI 模块

AI 模块位于 `src/main/ai/` 目录，在主进程中运行。

## 模块列表

| 文件 | 功能 | 依赖 |
|------|------|------|
| `ocr.ts` | 名片 OCR 识别 | Tesseract.js |
| `face.ts` | 人脸检测 | face-api.js |
| `text_analysis.ts` | 文本关键词提取和情感分析 | Node.js (`node-nlp`) |
| `intimacy.ts` | 亲密度自动计算 | 本地规则引擎 |
| `lost_contact.ts` | 断联检测 | 数据库查询 |
| `duplicate_detect.ts` | 重复联系人检测 | 本地相似度算法 |
| `smart_grouping.ts` | 智能分组 + Louvain 社区发现 | 自实现算法 |
| `suggestion_engine.ts` | 沟通建议引擎 | 规则引擎 |
| `personality_profile.ts` | 性格分析画像 | 统计分析 |
| `intimacy_prediction.ts` | 亲密度预测 | 线性回归 |
| `bridge_detector.ts` | 桥接人识别 | Betweenness Centrality |

## Tesseract.js OCR

- 用于名片扫描识别
- 支持中英文识别
- 从识别结果中提取姓名、电话、邮箱、公司、职位、地址

## face-api.js

- 基于 TensorFlow.js 的人脸检测
- 返回人脸位置坐标
- 生成人脸特征描述符

## 文本分析

- 中文分词 + 词频统计
- 基于词典的情感分析
- 正面/负面关键词提取

## Louvain 社区发现

- 自实现的 Louvain 算法
- 分析关系图结构
- 自动化分联系人社区
- 用于图谱增强的社区着色

## 智能分组

- 基于关系网络和交互频率
- 自动生成分组建议
- 分配分组颜色和名称

## 沟通建议引擎

- 分析交互历史
- 检测失联联系人
- 检查即将到来的生日/纪念日
- 生成分类建议（警告/信息/提示）

## 桥接人识别

- 计算 Betweenness Centrality
- 识别社交网络中的关键枢纽人物
- 衡量联系人连接不同社交圈的能力

## Ollama 集成

RelMap 支持检测本地 Ollama 实例，用于提供语义搜索等自然语言处理功能。Ollama 集成是可选的，应用在无 Ollama 环境下仍然可以正常运行。
