# AI 功能

RelMap 提供多种 AI 驱动的智能分析功能，帮助你更好地管理联系人关系。

## OCR 名片识别

使用 Tesseract.js 进行光学字符识别，从名片图片中提取信息：

- 姓名
- 手机号
- 邮箱
- 公司
- 职位
- 地址

自动将识别结果填入联系人表单，减少手动输入。

## 人脸检测

使用 face-api.js 检测照片中的人脸：

- 返回人脸位置和边界框
- 提供人脸特征描述符
- 可用于照片管理中的联系人标识

## 文本分析

### 关键词提取

对文本进行中文分词和词频统计，提取关键词语。

### 情感分析

分析文本的情感倾向：

- 返回情感分数（范围）
- 分类：positive / neutral / negative
- 输出正面/负面关键词列表

### 日记分析

对日记内容进行综合分析，同时输出关键词和情感分析结果。

## 智能建议

基于联系人的交互历史数据生成沟通建议：

- 长期未联系警告
- 纪念日/生日提醒
- 关系维护建议

建议类型分为：`warning`（警告）、`info`（信息）、`tip`（提示）

## 性格画像

基于交互记录分析联系人的性格画像：

- 主导沟通目的
- 情感基调评分
- 沟通风格分布
- 关系深度评分
- 情感变化趋势

## 亲密度预测

基于历史交互数据预测亲密度变化趋势：

- 当前亲密评分
- 30 天预测分数
- 90 天预测分数
- 趋势方向（up / stable / down）
- 预测置信度

## 断联检测

自动检测长时间未联系的联系人，可自定义时间阈值（月数）。

## 重复检测

添加新联系人时自动检测可能重复的已有联系人，基于姓名、公司、手机号、邮箱相似度。

## 智能分组

基于联系人的关系和交互数据，自动生成分组建议：

- 建议分组名称
- 建议分组颜色
- 建议成员列表
- 分组理由
- 相似度评分

## Ollama 集成

自动检测本地 Ollama 实例，用于语义搜索等功能。如果本地安装了 Ollama，RelMap 会自动发现可用的模型。
