# 指南

RelMap（关系管理地图）是一款本地优先、隐私安全的智能联系人关系管理应用。它帮助你管理联系人、可视化关系网络，并通过 AI 技术提供智能分析。

## 核心功能

- **联系人管理** — 记录联系人的详细信息、社交账号、分组、标签等
- **关系图谱** — 可视化展示联系人之间的关系网络，支持社区发现和导出
- **AI 分析** — OCR 名片识别、人脸检测、文本分析、智能建议、性格画像
- **日记与事件** — 记录与联系人相关的事件和日记，追踪互动历史
- **提醒与跟进** — 生日提醒、纪念日提醒、跟进任务管理
- **数据安全** — 本地存储、PIN 锁加密、BYOK 可选
