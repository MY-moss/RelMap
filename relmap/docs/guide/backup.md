# 备份与导入导出

RelMap 提供完整的数据备份和迁移方案。

## 备份

### 导出备份

将当前数据库导出为备份文件：

- 支持选择保存路径
- 可选加密密码保护
- 加密备份文件扩展名为 `.enc`
- 非加密备份文件扩展名为 `.db`

### 导入备份

从备份文件恢复数据：

- 支持 `.db` 和 `.enc` 格式
- 加密备份需要输入正确的密码
- 通过文件对话框选择备份文件

### 历史备份

应用会自动管理备份文件列表，方便查看和管理历史备份记录。

## 导入导出

### 导入

支持从 vCard 文件批量导入联系人：

- `importVCard(vcardText)` — 从 vCard 文本导入
- `importVCardFile()` — 通过文件对话框选择 vCard 文件导入
- 返回导入结果（成功数、跳过数、错误列表）

### 导出

支持多种格式导出数据：

- **CSV 导出** — 导出联系人列表为 CSV 格式
- **JSON 导出** — 可选择导出模式：
  - `contacts` — 仅导出联系人数据
  - `all` — 导出所有数据（联系人、关系、事件、日记等）
