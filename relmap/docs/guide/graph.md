# 关系图谱

关系图谱是 RelMap 的核心功能，以可视化方式展示联系人之间的关系网络。

## 基础图谱

图谱由节点和边组成：

- **节点** — 每个联系人是一个节点，节点大小反映亲密度
- **边** — 联系人之间的关系连线，边的粗细反映亲密度
- **颜色** — 收藏联系人有特殊高亮

### 交互操作

- 拖拽节点调整布局
- 滚轮缩放
- 悬停查看关系详情
- 点击节点查看联系人详情
- 亲密度筛选滑块，过滤低亲密度的关系

## 增强功能

### 节点详情

点击图谱中的节点，可查看：

- 联系人完整信息
- 与该节点相关的所有关系
- 节点所属社区

### Louvain 社区发现

基于 Louvain 算法自动识别联系人社区，相同社区的联系人使用相同颜色标识，帮助发现社交圈。

社区算法分析所有关系网络的连接密度，将联系紧密的群体自动聚合成社区。

### 亲密度分布

查看所有关系的亲密度统计分布，了解整体关系网络的亲密程度。

### 图谱导出

支持多种格式导出图谱数据：

| 格式 | 说明 |
|------|------|
| **PNG** | 导出为图片，方便分享和展示 |
| **JSON** | 完整的图谱数据结构，可用于数据分析 |
| **CSV** | 关系列表，可在表格软件中打开 |
| **.relgraph** | 图谱快照，可用于分享和重新导入 |

## 桥接人识别

自动识别在社交网络中起到桥梁作用的关键联系人（高 Betweenness Centrality 得分），这些人是连接不同社交圈的关键节点。
